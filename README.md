## 📦 Sistema de Gerenciamento de Estoque
 Este projeto é um sistema de gerenciamento de estoque, desenvolvido para a disciplina de Pensamento Computacional do curso de Ciências da Computacão. O programa permite ao usuário gerenciar produtos em um inventário.

# 🛠️ Tecnologia Utilizada 
Linguagem: Python (manipulação de dados, funções, listas, loops, etc...)
(Utilizamos as bibliotecas "Customtkinter" para criar uma interface, e "CTkMessagebox" para os pop-ups.

# Pré-requisitos
Antes de começar, você precisará ter o Python instalado em seu computador.

Você pode baixar o Python em: python.org/downloads

Nota: Recomenda-se usar a versão 3.x ou superior.

# 🚀 Modo de Usar:

Execute o arquivo principal `main.py` usando o Python para executar o programa no terminal.
Pronto! O menu do sistema de estoque deve aparecer no seu terminal. Siga as instruções na tela para cadastrar, consultar ou gerenciar produtos.

Para utilizar a interface, execute o arquivo `interface.py`.
